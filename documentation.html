Editing Content

To edit text content, simply open any HTML file in a code editor (like VS Code, Sublime Text, or Atom) and modify the text between the HTML tags.

Example: Changing a Heading

<!-- Original -->
<h1>We build digital experiences that drive growth.</h1>

<!-- Modified -->
<h1>Your New Agency Headline Here</h1>

Example: Changing Paragraph Text

<p class="lead">Creativo is a strategic digital agency...</p>

<p class="lead">Your agency description goes here...</p>

Tips:

  • Keep paragraphs concise for better readability
  • Maintain the existing CSS classes (like lead, btn, js-scroll)
  • Update all pages with your new content

Changing Colors

The template uses CSS variables for easy color customization. All colors are defined in the :root section of style.css.

:root {
    --color-primary: #4f46e5; /* Main brand color */
    --color-primary-dark: #4338ca;
    --color-dark: #111827; /* Text color */
    --color-gray-800: #1f2937;
    --color-gray-600: #4b5563;
    --color-gray-300: #d1d5db;
    --color-gray-100: #f9fafb;
    --color-white: #ffffff;
}

Current Color Scheme:

Primary (#4f46e5)
Primary Dark (#4338ca)
Text (#111827)
Light Gray (#f9fafb)

To change the primary color, simply update the --color-primary and --color-primary-dark values with your preferred colors.

Customizing Images

Replace the existing images with your own by following these steps:

  1. Prepare your images (optimize for web, recommended size: 1200x800px for portfolio images)
  2. Replace the files in the assets/images/ folder
  3. Keep the same file names, OR update the HTML to reference your new file names

Image Replacement Example:

<!-- Original -->
<img src="./assets/images/hero.png" alt="Creative workspace">

<!-- Updated with new image -->
<img src="./assets/images/my-new-hero.jpg" alt="Your agency workspace">

Recommended Image Specifications:

  • Hero images: 1200x800px, web-optimized
  • Portfolio images: 800x600px, consistent aspect ratio
  • Team photos: 400x400px, square format
  • Format: JPEG or PNG (PNG for transparency)
  • File size: Keep under 200KB per image

Animations

The template uses GSAP (GreenSock Animation Platform) with ScrollTrigger for subtle, professional animations.

How Animations Work:

Elements with the class js-scroll will fade up when they enter the viewport. The animations are controlled in assets/js/main.js.

// Scroll reveal animation setup
const scrollElements = document.querySelectorAll('.js-scroll');

// Elements fade up when 85% visible
const elementInView = (el) => {
  const elementTop = el.getBoundingClientRect().top;
  return (
    elementTop <= (window.innerHeight * 0.85)
  );
};

To Disable Animations:

Remove the js-scroll class from elements, or comment out the animation code in main.js.

To Add Animation to New Elements:

Simply add the js-scroll class to any element you want to animate on scroll.

<!-- This element will fade up on scroll -->
<div class="js-scroll">
  <h3>Your New Section</h3>
  <p>This content will animate into view.</p>
</div>

Responsive Design

The template is fully responsive and uses:

  • Bootstrap 5 Grid System (for layout only)
  • CSS Media Queries (in style.css)
  • Mobile-first approach

Breakpoints Used:

/* Large devices (desktops, 992px and up) */
@media (max-width: 992px) { ... }

/* Medium devices (tablets, 768px and up) */
@media (max-width: 768px) { ... }

/* Small devices (phones, 576px and up) */
@media (max-width: 576px) { ... }

Testing Responsiveness:

  • Resize your browser window to test different screen sizes
  • Use browser developer tools (F12) to simulate mobile devices
  • Test on actual mobile devices when possible
